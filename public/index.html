Projeto Sistema gestão escolar


Este projeto tem como ideal fazer uma base de um sistema de gestão escolar onde é possivel o aluno consultar suas notas, faltas, horário e solicitar documentos diretamente a secretaria da escola.

Também será criado uma página com interface similar, porém com opções diferentes no menu para que os doscentes possam fazer a gestão e atualizem os dados.
